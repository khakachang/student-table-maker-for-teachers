# Student Table Maker for teacher
Easily Make Student table like test marks with their name, roll. no marks they obtain, additional features: show a graph too average marks, highest marks, lowest
**App Flow:**
Teacher open the app
Add respective columns (they can also remove the columns)
And then the UI will be updated accordingly to fill those columns
save the file as PDF,.txt files
