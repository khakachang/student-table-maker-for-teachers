# Student Table Maker for Teachers

A desktop application designed to help teachers easily create and manage student records such as test marks and performance data.

## Features
- Create student tables with fields like **Name, Roll Number, Marks**, and other customizable columns  
- Add or remove columns dynamically based on requirements  
- Interactive UI that updates automatically when columns are modified  
- Generate performance insights:
  - Average marks  
  - Highest marks  
  - Lowest marks  
  - Graphical representation of data  
- Export data to files:
  - PDF format  
  - Text (.txt) format  

## Application Flow
1. Teacher opens the application  
2. Adds or customizes table columns  
3. UI updates dynamically based on selected columns  
4. Teacher enters student data  
5. Views performance analysis and graphs  
6. Saves or exports the data as PDF or text file
